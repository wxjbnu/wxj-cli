## 使用说明
- npm install -g wxj-cli (已完成)
- wxj-cli create (template) (name) "template是github上面的user/repo ,name是文件夹名 (已完成)
- cd (name) && wxj-cli compile (type) 将当前模板编译成目标模板,type可以为vue,react,wxxcx.... (未开发)
